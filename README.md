# 待办清单

一个简洁美观的待办清单网页应用。

## 功能特性

- ✅ 添加新任务（支持回车键快速添加）
- ✅ 点击任务标记完成（划线效果）
- ✅ 删除任务
- ⭐ 设置优先级（重要任务红色标签显示，排在前面）
- ⏰ 截止时间（设置倒计时，显示进度条）
- 📊 进度条（蓝色正常，橙色接近截止，红色已过期）
- 📁 分组显示（进行中 / 已完成）
- 🔄 折叠已完成任务
- 💾 数据持久化（localStorage）
- 📥 导入/导出任务（支持覆盖或合并）
- 🎨 蓝色系现代化 UI 设计
- 📱 响应式布局，自适应滚动

## 使用方法

直接用浏览器打开 `todo-list/index.html` 即可使用。

### 添加任务

1. 在输入框输入任务内容
2. 点击 ⭐ 按钮设置重要（可选）
3. 点击「添加」按钮或按回车

### 设置截止时间

1. 点击任务右侧的时钟图标
2. 输入分钟数：
   - 30 = 30分钟
   - 60 = 1小时
   - 1440 = 1天
   - 10080 = 1周
3. 输入 0 可清除截止时间

### 导入/导出

1. 点击右上角设置按钮
2. 导出任务：生成 JSON 文件
3. 导入任务：选择覆盖（替换所有）或合并（添加到现有）

## 技术栈

- HTML5
- CSS3（Flexbox 布局、渐变、动画）
- JavaScript（原生，无框架）
- localStorage 本地存储
