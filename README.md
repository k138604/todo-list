# 待办清单

一个简洁美观的待办清单网页应用。

## 功能特性

- ✅ 添加新任务（支持回车键快速添加）
- ✅ 点击任务标记完成/未完成（划线效果）
- ✏️ 编辑任务（点击编辑按钮直接修改任务文本）
- 🗑️ 删除单个任务
- ⭐ 设置优先级（重要任务红色标签显示，排在前面）
- ⏰ 截止时间（设置倒计时，显示剩余时间）
- 📊 进度条（蓝色正常，橙色接近截止，红色已过期）
- 📁 分组显示（进行中 / 已完成）
- 🔄 折叠/展开已完成任务
- 📊 任务计数（实时显示进行中/已完成数量）
- 💾 数据持久化（localStorage 自动保存）
- 📥 导入/导出任务（支持覆盖或合并）
- 🗑️ 一键清空所有任务
- 🎨 蓝色系现代化 UI 设计
- 📱 响应式布局，自适应滚动

## 使用方法

直接用浏览器打开 `todo-list/index.html` 即可使用。

### 添加任务

1. 在输入框输入任务内容
2. 按回车或点击「添加」按钮

### 编辑任务

1. 点击任务右侧的编辑图标（铅笔）
2. 直接修改任务文本
3. 按回车或点击其他地方保存

### 完成任务

- 点击任务文本即可标记完成/未完成
- 已完成任务显示划线效果，可折叠隐藏

### 设置截止时间

1. 点击任务右侧的时钟图标
2. 输入分钟数：
   - 30 = 30 分钟
   - 60 = 1 小时
   - 1440 = 1 天
   - 10080 = 1 周
   - 0 = 清除截止时间
3. 进度条实时显示剩余时间

### 设置重要

- 点击任务星形图标标记为重要
- 重要任务显示红色标签，优先排序

### 导入/导出

1. 点击右上角设置按钮
2. 导出任务：生成 JSON 文件下载
3. 导入任务：选择 JSON 文件，确认覆盖或合并

### 清空任务

1. 点击右上角设置按钮
2. 点击「清空全部」按钮
3. 确认后删除所有任务（不可恢复）

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Enter` | 添加任务 / 确认编辑 |
| `ESC` | 关闭模态框 |

## 技术栈

- **HTML5** - 语义化结构
- **CSS3** - Flexbox 布局、渐变、动画、自定义滚动条
- **JavaScript** - 原生 ES5+，无框架依赖
- **localStorage** - 本地数据持久化

## 浏览器支持

- Chrome / Edge（推荐）
- Firefox
- Safari
- 其他支持 ES5 和 localStorage 的现代浏览器
